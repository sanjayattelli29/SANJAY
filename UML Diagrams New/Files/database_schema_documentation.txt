CUSTOMER (Role: Customer in AspNetRoles)
- Primary User: Person who wants to buy insurance
- Database Columns: Id, FullName, Email, PhoneNumber, AadhaarNumber
- Can: Register, Apply for policies, File claims, Chat with agents, View notifications

AGENT (Role: Agent in AspNetRoles)
- Primary User: Employee who approves policy applications
- Database Columns: Id, FullName, Email, PhoneNumber, BankAccountNumber, InitialPassword
- Can: Review applications, Approve/reject policies, Chat with customers, View analytics

CLAIM OFFICER (Role: ClaimOfficer in AspNetRoles)
- Primary User: Employee who processes insurance claims
- Database Columns: Id, FullName, Email, PhoneNumber, BankAccountNumber, InitialPassword
- Can: Review claims, Approve/reject claims, Review documents, Add remarks

ADMIN (Role: Admin in AspNetRoles)
- Primary User: System administrator
- Database Columns: Id, FullName, Email, PhoneNumber
- Can: Create agents/officers, Manage users, View dashboard, Generate reports, Assign tasks
